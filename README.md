NYC Airbnb Market Analysis

Tools: Python · pandas · SQL (MySQL) · Matplotlib · Seaborn · Tableau

Dataset: NYC Airbnb Open Data — Kaggle

Tableau Dashboard: https://public.tableau.com/authoring/NYCAirbnbMarketAnalysis_17814049369560/Dashboard1#2

Project Overview
Analyzed 48,645 Airbnb listings across New York City to uncover pricing patterns,
neighborhood trends, and availability insights. The project covers the full data
analyst workflow — data cleaning, exploratory analysis, SQL querying in MySQL
Workbench, Python visualization, and an interactive Tableau dashboard.

Key Findings

Manhattan dominates with 21,488 listings and the highest average nightly
price at $179 — nearly double the Bronx ($85) and above Brooklyn ($118).
Entire home/apt listings average $195/night — more than 2× the cost of
private rooms ($85) and shared rooms ($68).
Price sensitivity sweet spot: Manhattan entire-home listings priced
$120–$180/night receive ~27% more reviews than listings above $250, suggesting
higher occupancy at mid-range prices.
Brooklyn is the volume runner-up with 20,041 listings at a lower average
price ($118), making it the best-value borough for guests.
Tribeca is the priciest neighbourhood at $330/night average, followed by
NoHo ($276) and Flatiron District ($275).
